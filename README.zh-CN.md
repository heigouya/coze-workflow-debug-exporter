# Coze Workflow Debug Exporter

这是一个 Chrome 扩展，用来抓取 Coze / 扣子工作流试运行时前端接口返回的数据，并整理成适合发给 AI 排查问题的文本。

## 安装

1. 下载或克隆本仓库。
2. 打开 Chrome，进入 `chrome://extensions/`。
3. 打开右上角「开发者模式」。
4. 点击「加载已解压的扩展程序」。
5. 选择本仓库根目录。
6. 打开 Coze 工作流页面，重新试运行一次工作流。
7. 点击浏览器右上角扩展图标，选择「打开调试报告」。

## 使用

- 报告页会按运行批次展示抓到的数据。
- 每张卡片代表一个节点或一个原始接口响应。
- 报错节点会用红色边框突出。
- 「复制本节点」只复制当前卡片。
- 「一键复制全部」会把本次运行的节点数据按顺序整理成一段文本，方便直接发给 AI。

## 注意

- 扩展只能抓到安装并启用之后，Coze 页面实际请求过的数据。
- 数据保存在本机 Chrome 扩展本地存储里，可以在弹窗中清空。
- 复制文本时默认会脱敏 token、secret、cookie、签名 URL 参数等明显敏感字段。
